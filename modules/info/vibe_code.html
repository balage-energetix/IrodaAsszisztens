Developer Tutorial V1.0

Üdvözöllek a Vájb Kód világában! Ez a modul lépésről lépésre végigvezet az Antigravity AI telepítésén, a modern webes fejlesztés alapjain és a projekted világméretű közzétételén.

1. Kezdés az Antigravity-vel

Az Antigravity egy forradalmi AI kódoló asszisztens, amely közvetlenül a fájljaid között dolgozik, külön szerkesztő telepítése nélkül is használható.

01 Nyisd meg a böngésződben az Antigravity felületét.
02 Csatold a munkakörnyezetedet (Workspace) az AI-hoz.
03 Hitelesítés után egyszerűen csak írd le magyarul: "Hozz létre egy új modult!"
04 Az AI azonnal generálja a kódot, amit rögtön tesztelhetsz is.

2. Interaktív Játszótér: CSS Élőben

Próbáld ki, hogyan változik egy elem megjelenése a kód módosításával. Szerkeszd a bal oldali mezőt!

C) JS (Logika): Ez teszi interaktívvá az oldalt (pl. mentés).
function save() { localStorage.setItem('myNote', document.getElementById('note').value); }

3. Hosting a GitHub-on (Élesítés)

Hogy mások is láthassák a munkádat, a GitHub Pages a legegyszerűbb megoldás.

Lépés 1 Regisztrálj a github.com oldalon és hozz létre egy új "Repository"-t (pl. irodai-asszisztens).
Lépés 2 Töltsd fel a fájljaidat (Upload files), ügyelve rá, hogy legyen egy index.html a főkönyvtárban.
Lépés 3 Menj a Settings -> Pages menüpontba.
Lépés 4 Válaszd ki a 'main' branch-et forrásnak, és ments rá.
Eredmény 1-2 perc múlva az oldalad elérhető lesz a https://felhasznalonev.github.io/irodai-asszisztens/ címen!
PRO TIPP: Használd az Antigravity-t a kódolás közben: "Húzz egy üveg-effektust a kártyákra!" - és ő megcsinálja helyetted a CSS-t.
